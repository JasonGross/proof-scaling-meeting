| 1:30 - 2:00   | FVAPPS (Ronak Mehta and Quinn Dougherty)                    | <details><summary>Abstract</summary> We introduce the Formally Verified Automated Programming Progress Standards, or FV-APPS, a benchmark for writing programs and proving their correctness. Previously, APPS provided a benchmark and dataset for programming puzzles to be completed in python and checked against unit tests, of the kind seen in technical assessments in the software engineering industry. We generalize the unit tests to Lean 4 theorems, given without proof (Lean's "sorry" keyword), and challenge the machine learning and program synthesis communities to solve both each general purpose programming problem and it's associated correctness specifications. It is available for play at https://huggingface.co/datasets/quinn-dougherty/fvapps </details> |
